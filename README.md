# yatagarrage

30種類以上の弾幕パターンを搭載した本格派ブラウザ弾幕シューティング

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 特徴

- **30種類以上の弾幕パターン** - 円形、螺旋、波状、ホーミングなど、著名な弾幕シューティングでお馴染みのパターンを完全再現
- **クロスプラットフォーム対応** - PC、スマホ、タブレットで快適にプレイ可能
- **美麗なエフェクト** - 残像エフェクトと重力場による弾道変化で臨場感あふれる演出
- **直感的な操作** - キーボードでもタッチでも簡単操作

## 操作方法

### PC（キーボード）

- **矢印キー**: プレイヤーを左右に移動
- **スペースキー**: 弾を発射

### スマホ・タブレット（タッチ操作）

- **画面中央をタップ**: プレイヤーがタップした位置に移動
- **画面外周をタップ**: タップした方向に弾を発射

タップ操作は直感的で、移動と攻撃を同時に行えます！

## ゲームの楽しみ方

### 基本ルール

1. 青い三角形（自機）を操作して敵を撃破
2. 敵や敵の弾に当たるとゲームオーバー
3. できるだけ高得点を目指そう

### 攻略のコツ

- **敵の弾幕パターンを見極める** - 30種類以上のパターンがランダムで出現します
- **重力場を活用** - 弾道が曲がる重力場を利用して敵弾を回避
- **残像を見て軌道予測** - すべてのオブジェクトに残像が表示されるので、次の動きを予測しやすい

## 弾幕パターン

このゲームには著名な弾幕シューティングでよく見られる以下のパターンが実装されています：

- **円形弾幕系**: 8方向、16方向、24方向、32方向
- **螺旋弾幕系**: シングル、ダブル、トリプルスパイラル
- **特殊軌道系**: 波状、カーブ、ホーミング、加速・減速
- **幾何学系**: 十字型、X字型、米字型、花弁状
- **複合系**: 二段階、時間差、収束拡散、ランダム

それぞれのパターンは東方Project、怒首領蜂、虫姫さまなどの名作から着想を得ています。
